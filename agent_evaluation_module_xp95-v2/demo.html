Eval agent · blind verdict?
A third LLM (different vendor than the agent model so judge family-bias is minimized) reads both responses in randomized A/B order with NO label indicating which received the harness. It scores each across the configured dimensions (default: Safety, Reasoning depth, Seriousness, Tone appropriateness) on a 0-10 scale, picks a stronger answer, and writes a 1-line reason. The shuffle randomization seed is per-run and not logged to the judge, so harness-bias is genuinely blind.

Cognitive posture field?
Three visualizers in this fieldset, all driven by heuristic text analysis (no logprobs, no second LLM call). 1. CUBES — per-chunk CONTENT patterns. Four 10×10 grids per agent. The response is sliced into 100 chunks; each grid analyses one signal per chunk, then a 2D Gaussian blur turns sparse markers into soft halos. • CONFIDENCE (diverging blue↔red) — hedge words (maybe, might, seems) vs assertive (must, always, definitely) plus comma/period cadence. • REASONING (dark→hot orange) — explicit logical connectives (because, therefore, since, thus, if/then). • SYCOPHANCY (dark→magenta) — capitulation, agreement, validation drift, approval-seeking. • EVIDENCE (dark→bright green) — anchored claims ("based on", "the data shows", "specifically", "according to"). The harness agent typically shows denser reasoning + evidence, lower sycophancy, less aggressive confidence. 2. SPECTROMETER — per-position STRUCTURAL patterns. Four rows per agent, X-axis = response position (40 chunks), Y-axis = signal strength. • PACE (violet) — mean words-per-sentence per chunk. Winding = bad. • CONC (teal) — concretion: numbers, percentages, codes, URLs, named entities. Concrete = good. • WAFL (gold) — waffle: transitional padding ("basically", "essentially", "to be clear"). Padded = bad. • REPT (crimson) — repetition: TF-IDF cosine overlap with adjacent chunks. Looping = bad. Harness story: low PACE / high CONC / low WAFL / low REPT. 3. DIFF STRIP — per-position B−A delta. Single signed bar chart between the two agents' spectrometers. For each chunk: positive (blue, bar up) = harness ate failure modes here; negative (red, bar down) = raw was better here. Aggregates all four spectrometer dimensions: PACE/WAFL/REPT counted as negative-for-harness, CONC counted as positive-for-harness.

Reasoning topology · node-by-node trace?
The structured cognitive operation the harness handed Agent B. Each node is one step of the procedural reasoning. Edges show flow between nodes. After Agent B's response streams in, nodes whose content overlaps with the response text light up via heuristic keyword match (this is a heuristic visualization, not ground truth of which path the model actually took). Edge types: green = "yes" branch from a gate, red = "no" branch, cyan = M-node "working" (reasoning healthy), dashed red = M-node "failing" (reasoning broken), purple dotted = LOOP back-edge, gold dashed = RE-ENTER jump.
