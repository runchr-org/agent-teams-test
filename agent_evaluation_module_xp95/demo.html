Cognitive posture field?
Four 10x10 grids. Two per agent. Computed from the response text alone, no logprobs needed. CONFIDENCE (top grids) - diverging palette, hedged blue to assertive red. Each cell counts hedge words (maybe, might, seems, could, would, slightly) vs assertive words (must, always, definitely, clearly, completely) in its slice of the response. Punctuation cadence adds signal too: commas lean hedged, periods lean assertive. Heat zones show where the response leans cautious vs confident. REASONING (bottom grids) - sequential palette, dark to hot. Each cell counts explicit reasoning connectives (because, therefore, since, thus, if/then, due to, as a result, this means). Bright zones show where the response is visibly reasoning rather than just declaring. A 2D Gaussian blur is applied so a single marker bleeds into a soft halo and sparse signals form organic shapes rather than isolated pixels. This is a second signal, independent of the judge. The judge scores content; these grids show posture. On adversarial / emotionally loaded prompts, the harness agent typically shows a denser reasoning grid and a less assertive confidence grid than the raw agent. That's the visual signature of the harness biting.
